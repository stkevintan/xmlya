# Xmlya Project

A project about Ximalaya(喜马拉雅) FM: 

1. Ximalaya sdk: an unofficial sdk for ximalaya
2. Ximalaya vscode extension: a tiny but powerful extension which can help you listen to Ximalaya FM in vscode
